# API Reference

Documentation for JSON Resume API endpoints.

## Base URL

```
https://registry.jsonresume.org
```

## Endpoints

### Get Resume

```http
GET /{username}
```

Fetch a user's resume from their GitHub Gist.

**Parameters:**

- `username` (path) - GitHub username

**Example:**

```bash
curl https://registry.jsonresume.org/thomasdavis
```

### Theme Preview

```http
GET /{username}?theme={theme-name}
```

Render resume with a specific theme.

**Parameters:**

- `username` (path) - GitHub username
- `theme` (query) - Theme name (e.g., "professional", "elegant")

**Example:**

```bash
curl https://registry.jsonresume.org/thomasdavis?theme=professional
```
