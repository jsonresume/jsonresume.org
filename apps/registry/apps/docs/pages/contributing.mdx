# Contributing

Thank you for your interest in contributing to JSON Resume!

## Getting Started

1. Fork the repository
2. Clone your fork
3. Install dependencies: `pnpm install`
4. Create a branch: `git checkout -b feature/your-feature`
5. Make your changes
6. Run tests: `pnpm test`
7. Commit: `git commit -m "feat: your feature"`
8. Push: `git push origin feature/your-feature`
9. Open a Pull Request

## Development Setup

```bash
# Clone the repository
git clone https://github.com/jsonresume/jsonresume.org.git
cd jsonresume.org

# Install dependencies
pnpm install

# Start the registry app
cd apps/registry
pnpm dev
```

## Code Standards

- Maximum 200 lines per file (production code)
- Single Responsibility Principle
- Comprehensive documentation
- Test critical functionality

## Testing

```bash
# Run unit tests
pnpm test

# Run E2E tests
pnpm test:e2e

# Run with coverage
pnpm test:coverage
```
