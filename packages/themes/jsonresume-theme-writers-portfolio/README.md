# Writer's Portfolio

Literary, elegant theme with transitional serif typography. Reads like prose with larger body font for readability.

![Writer's Portfolio theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/writers-portfolio.png)

**Tags:** writer, literary, elegant, minimal, serif

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=writers-portfolio

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-writers-portfolio
resume export resume.html --theme writers-portfolio
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-writers-portfolio build
```
