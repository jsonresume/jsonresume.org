# Clinical Precision

Clean, clinical, calm resume styled like a medical chart record. Each entry is a chart row with a green-railed vital column of monospaced date codes, lab-panel skill cards, and a "chart note" summary. Clinical green accents on white with mint tints. Tailored for doctors, nurses, and healthcare administrators.

![Clinical Precision theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/clinical-precision.png)

**Tags:** healthcare, clinical, medical, green, chart, calm, professional

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=clinical-precision

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-clinical-precision
resume export resume.html --theme clinical-precision
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-clinical-precision build
```
