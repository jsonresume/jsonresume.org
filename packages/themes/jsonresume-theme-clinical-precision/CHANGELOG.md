# jsonresume-theme-clinical-precision

## 0.2.0

### Minor Changes

- 4715518: Add Clinical Precision theme — a clean, calm resume styled like a medical chart record. Each experience, education, project, and volunteer entry renders as a "chart row" with a green-railed vital column holding monospaced IBM Plex Mono date codes and location/area metadata, paired with a content column. Skills appear as lab-panel cards, the summary is framed as a "chart note", and section titles carry monospace chart codes. Clinical green (#059669) accents on white with soft mint tints. Renders all JSON Resume sections and is print-optimized. Tailored for doctors, nurses, and healthcare administrators.
