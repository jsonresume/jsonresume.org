# Executive Slate

Authoritative, polished theme with serif headings and modern sans body. Classic print-like layout with elegant type hierarchy.

![Executive Slate theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/executive-slate.png)

**Tags:** executive, professional, senior, slate, premium

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=executive-slate

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-executive-slate
resume export resume.html --theme executive-slate
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-executive-slate build
```
