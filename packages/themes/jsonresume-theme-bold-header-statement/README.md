# Bold Header Statement

Magazine-inspired layout with commanding header and confident personal brand aesthetic.

![Bold Header Statement theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/bold-header-statement.png)

**Tags:** bold, header, magazine, personal-brand, creative, modern

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=bold-header-statement

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-bold-header-statement
resume export resume.html --theme bold-header-statement
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-bold-header-statement build
```
