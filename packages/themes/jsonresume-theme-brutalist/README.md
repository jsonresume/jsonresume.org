# jsonresume-theme-brutalist

Raw concrete brutalism for JSON Resume. Heavy 4px black rules, oversized
UPPERCASE Archivo Black / Space Grotesk headings, hard edges (zero rounded
corners), stark black-on-white with a single harsh safety-yellow accent,
monospace spec labels, numbered sections, and hard offset block-shadows that
make the structural grid literal.

## Usage

```bash
resume export resume.html --theme brutalist
```

Renders every JSON Resume section: basics/summary, work, education, skills,
projects, volunteer, awards, certificates, publications, languages, interests,
and references.
