# Mid-Century Resume

Retro modernist elegance inspired by 1960s design offices with vintage typography.

![Mid-Century Resume theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/mid-century-resume.png)

**Tags:** mid-century, retro, vintage, modernist, 1960s, geometric, creative

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=mid-century-resume

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-mid-century-resume
resume export resume.html --theme mid-century-resume
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-mid-century-resume build
```
