# New York Editorial

Magazine-quality editorial polish with tight column layout, serif headings, and typographic elegance. Inspired by The Atlantic.

![New York Editorial theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/new-york-editorial.png)

**Tags:** editorial, magazine, serif, literary, journalism, writer

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=new-york-editorial

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-new-york-editorial
resume export resume.html --theme new-york-editorial
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-new-york-editorial build
```
