# Modern Classic

Professional, timeless resume with humanist sans-serif typography. Print-optimized with perfect balance between aesthetics and ATS compliance.

![Modern Classic theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/modern-classic.png)

**Tags:** professional, timeless, modern, classic, blue

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=modern-classic

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-modern-classic
resume export resume.html --theme modern-classic
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-modern-classic build
```
