# Two-Column Modernist

Structured European minimalism with asymmetric two-column layout and neutral grotesk typography.

![Two-Column Modernist theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/two-column-modernist.png)

**Tags:** two-column, european, minimalist, structured, grotesk, swiss-design

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=two-column-modernist

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-two-column-modernist
resume export resume.html --theme two-column-modernist
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-two-column-modernist build
```
