# Architect's Portfolio

Structural elegance with horizontal keylines and geometric precision like a drafting sheet.

![Architect's Portfolio theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/architects-portfolio.png)

**Tags:** architect, portfolio, geometric, precision, designer, structural, minimal

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=architects-portfolio

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-architects-portfolio
resume export resume.html --theme architects-portfolio
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-architects-portfolio build
```
