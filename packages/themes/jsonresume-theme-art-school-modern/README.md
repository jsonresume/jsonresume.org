# Art School Modern

Avant-garde gallery aesthetic for designers and artists. Oversized Archivo Black display type, an asymmetric off-grid header with a framed photo plate, an editorial Fraunces serif statement, catalogue-numbered sections, and a single vivid vermilion accent.

![Art School Modern theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/art-school-modern.png)

**Tags:** art, gallery, avant-garde, editorial, expressive, designer, creative, portfolio, display-type

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=art-school-modern

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-art-school-modern
resume export resume.html --theme art-school-modern
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-art-school-modern build
```
