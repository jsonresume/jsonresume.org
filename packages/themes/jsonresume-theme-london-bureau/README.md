# London Bureau

Traditional professionalism with heritage feel. Modeled after Financial Times style with serif typography, tight columns, and consistent rule dividers.

![London Bureau theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/london-bureau.png)

**Tags:** traditional, heritage, professional, serif, editorial, classic, formal

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=london-bureau

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-london-bureau
resume export resume.html --theme london-bureau
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-london-bureau build
```
