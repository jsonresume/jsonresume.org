# Nordic Minimal

Scandinavian restraint with white space worship and quiet confidence. Two-column layout with wide margins and extra line height for air.

![Nordic Minimal theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/nordic-minimal.png)

**Tags:** nordic, minimal, scandinavian, two-column, clean, steel-blue

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=nordic-minimal

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-nordic-minimal
resume export resume.html --theme nordic-minimal
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-nordic-minimal build
```
