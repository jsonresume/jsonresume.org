# Sidebar Photo Strip

European creative layout with integrated grayscale photo sidebar and minimal aesthetic.

![Sidebar Photo Strip theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/sidebar-photo-strip.png)

**Tags:** sidebar, photo, european, creative, minimal, two-column, portrait

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=sidebar-photo-strip

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-sidebar-photo-strip
resume export resume.html --theme sidebar-photo-strip
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-sidebar-photo-strip build
```
