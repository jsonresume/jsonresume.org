# Asymmetric Timeline

Cinematic storytelling with staggered timeline layout and serif-sans typography mix. Experiences arranged in alternating left-right bands along a central rule.

![Asymmetric Timeline theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/asymmetric-timeline.png)

**Tags:** timeline, asymmetric, storytelling, project-driven, narrative, visual-flow

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=asymmetric-timeline

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-asymmetric-timeline
resume export resume.html --theme asymmetric-timeline
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-asymmetric-timeline build
```
