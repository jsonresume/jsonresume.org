# jsonresume-theme-art-deco

1920s Gatsby elegance for JSON Resume. A symmetric centered masthead on a deep
midnight-to-emerald jewel panel with metallic-gold accents, geometric chevron
and sunburst dividers, and a high-contrast Cinzel didone-style display serif
over a graceful Cormorant Garamond body. Ornamental but legible.

## Usage

```bash
resume export resume.html --theme art-deco
```

Renders every JSON Resume section: basics/summary, work, education, skills,
projects, volunteer, awards, certificates, publications, languages, interests,
and references.
