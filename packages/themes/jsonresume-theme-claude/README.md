# Claude

Clean, modern theme with gradient accent header, dark mode support, and inline SVG icons. Designed and built entirely by Claude Code.

![Claude theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/claude.png)

**Tags:** modern, clean, dark-mode, ai, gradient, responsive

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=claude

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-claude
resume export resume.html --theme claude
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-claude build
```
