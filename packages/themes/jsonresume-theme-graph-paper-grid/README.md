# Graph Paper Grid

Technical design precision with subtle graph paper grid background and monospace typography. Built for engineers, architects, and UX designers.

![Graph Paper Grid theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/graph-paper-grid.png)

**Tags:** technical, engineer, architect, grid, monospace, systematic, precision

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=graph-paper-grid

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-graph-paper-grid
resume export resume.html --theme graph-paper-grid
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-graph-paper-grid build
```
