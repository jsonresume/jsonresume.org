# Investor Brief

Crisp, analytical, results-focused theme resembling a fund report. Features Key Metrics section and data-first presentation.

![Investor Brief theme preview](https://raw.githubusercontent.com/jsonresume/jsonresume.org/master/apps/homepage2/public/img/themes/investor-brief.png)

**Tags:** analytical, professional, finance, metrics, corporate, data-driven, crisp, vc

## Use it

Preview it live on the JSON Resume registry:

- https://registry.jsonresume.org/thomasdavis?theme=investor-brief

Install and use it with the JSON Resume CLI:

```sh
npm install jsonresume-theme-investor-brief
resume export resume.html --theme investor-brief
```

## Sections

Renders all standard JSON Resume sections: work, education, skills, projects, volunteer, awards, publications, languages, interests, references.

## Development

This theme lives in the [jsonresume.org](https://github.com/jsonresume/jsonresume.org) monorepo. Build it from the repo root with:

```sh
pnpm --filter jsonresume-theme-investor-brief build
```
